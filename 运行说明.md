# DouyinCrawler 本地运行说明

## 项目概览

- **项目**：DouyinCrawler（抖音数据采集工具）
- **后端**：Python 3.12+ / FastAPI
- **前端**：React 18 + TypeScript + Vite
- **运行模式**：Web 服务
- **核心功能**：抖音热榜数据采集、视频评论分析、数据可视化大屏

## 一、环境要求

- **Python** 3.12+
- **Node.js**（用于构建前端，需 npm 或 pnpm）
- 可选：**uv**（推荐用于 Python 依赖）、**aria2**（用于下载）

## 二、运行步骤

在项目根目录（`douyin-main`）打开 PowerShell 或 CMD，按顺序执行：

### 方式 A：一键菜单（推荐）

```powershell
.\quick-start.ps1
```

在菜单中：
1. 首次使用：先选 **1** 配置 uv 环境（或自行用 pip 安装依赖）
2. 需要下载功能：选 **2** 下载 aria2
3. 选 **3** 构建开发环境（安装前端依赖并构建）
4. 选 **4** 运行应用

### 方式 B：手动命令

**1. 安装 Python 依赖**

```powershell
# 有 uv 时（推荐）
uv sync

# 或用 pip
pip install -r requirements.txt
```

**2. 构建前端**

```powershell
cd frontend
pnpm install
pnpm run build
# 没有 pnpm 可用：npm install && npm run build
cd ..
```

**3. 启动应用**

- **桌面窗口模式**（带 GUI）：
  ```powershell
  python main.py
  ```

- **仅 Web 服务**（浏览器访问 http://localhost:8000）：
  ```powershell
  python -m backend.server
  ```

 指定端口示例：
  ```powershell
  python -m backend.server --port 9000
  ```

- **前端开发模式**（实时预览，浏览器访问 http://localhost:5173）：
  ```powershell
  cd frontend
  npm run dev
  ```

## 三、核心功能说明

### 1. 抖音热榜数据采集
- 自动从抖音热榜页面采集数据
- 支持定时刷新和手动刷新
- 数据存储到 MySQL 数据库

### 2. 视频评论分析
- 支持从热榜项目中爬取对应视频的评论
- 提供情感分析、关键词提取等功能
- 分析结果可视化展示

### 3. 数据可视化大屏
- 热榜数据实时展示
- **视频点击跳转**：点击热榜项目可直接跳转到对应抖音视频
- 支持数据趋势分析和对比

### 4. 视频跳转功能
- 热榜项目点击后会自动识别视频 ID
- 如果存在 19 位视频 ID，跳转到抖音视频页面
- 如果没有视频 ID，跳转到热榜链接
- 确保跳转链接的正确性和有效性

## 四、常见问题

1. **端口被占用**：默认 8000，可在设置或 `backend.server` 的 `--port` 中修改。
2. **前端未构建**：必须先完成「构建前端」步骤，否则桌面版/Web 版无法加载页面。
3. **首次使用**：在应用设置中配置 Cookie 后即可采集数据。
4. **路径含中文**：若在终端中路径报错，可把项目复制到无中文路径（如 `D:\dev\douyin-main`）再执行上述命令。
5. **视频跳转失败**：确保数据库中存在有效的视频 ID（19 位数字），可通过刷新热榜数据解决。

## 五、CLI 命令行模式

仅使用爬虫、不需要界面时：

```powershell
python -m backend.cli -u https://www.douyin.com/user/xxx -l 20
```

更多用法见 [USAGE.md](USAGE.md)。
