# DouyinCrawler 本地运行说明

## 项目概览

- **项目**：DouyinCrawler（抖音数据采集工具）
- **后端**：Python 3.12+ / FastAPI
- **前端**：React 18 + TypeScript + Vite
- **运行模式**：桌面 GUI（PyWebView）或 纯 Web 服务

## 一、环境要求

- **Python** 3.12+
- **Node.js**（用于构建前端，需 npm 或 pnpm）
- 可选：**uv**（推荐用于 Python 依赖）、**aria2**（用于下载）

## 二、运行步骤

在项目根目录（`douyin-main`）打开 PowerShell 或 CMD，按顺序执行：

### 方式 A：一键菜单（推荐）

```powershell
.\quick-start.ps1
```

在菜单中：
1. 首次使用：先选 **1** 配置 uv 环境（或自行用 pip 安装依赖）
2. 需要下载功能：选 **2** 下载 aria2
3. 选 **3** 构建开发环境（安装前端依赖并构建）
4. 选 **4** 运行应用

### 方式 B：手动命令

**1. 安装 Python 依赖**

```powershell
# 有 uv 时（推荐）
uv sync

# 或用 pip
pip install -r requirements.txt
```

**2. 构建前端**

```powershell
cd frontend
pnpm install
pnpm run build
# 没有 pnpm 可用：npm install && npm run build
cd ..
```

**3. 启动应用**

- **桌面窗口模式**（带 GUI）：
  ```powershell
  python main.py
  ```

- **仅 Web 服务**（浏览器访问 http://localhost:8000）：
  ```powershell
  python -m backend.server
  ```

 指定端口示例：
  ```powershell
  python -m backend.server --port 9000
  ```

## 三、常见问题

1. **端口被占用**：默认 8000，可在设置或 `backend.server` 的 `--port` 中修改。
2. **前端未构建**：必须先完成「构建前端」步骤，否则桌面版/Web 版无法加载页面。
3. **首次使用**：在应用设置中配置 Cookie 后即可采集数据。
4. **路径含中文**：若在终端中路径报错，可把项目复制到无中文路径（如 `D:\dev\douyin-main`）再执行上述命令。

## 四、CLI 命令行模式

仅使用爬虫、不需要界面时：

```powershell
python -m backend.cli -u https://www.douyin.com/user/xxx -l 20
```

更多用法见 [USAGE.md](USAGE.md)。
