# 抖音热榜评论定时爬取器 - 使用指南

## 🎯 功能特点

- ✅ **自动爬取**：每 2 小时自动执行一次
- ✅ **无需询问**：完全自动化运行
- ✅ **自动保存**：评论数据自动保存到 CSV
- ✅ **日志记录**：详细记录每次爬取结果
- ✅ **后台运行**：支持 Windows 任务计划

## 🚀 快速开始

### 方法 1：直接运行（推荐）

双击运行批处理文件：

```bash
start_scheduler.bat
```

**说明：**
- 会在当前目录打开命令行窗口
- 按 Ctrl+C 停止
- 适合临时使用

### 方法 2：后台服务运行

使用 Python 直接运行：

```bash
python scheduler_hot_comment.py --interval 2
```

**参数说明：**
- `--interval 2`：每 2 小时执行一次
- `--video-count 10`：每次爬取 10 个热榜视频
- `--comments-per-video 100`：每个视频爬取 100 条评论
- `--run-once`：只运行一次（测试用）

### 方法 3：Windows 任务计划程序（推荐长期使用）

#### 步骤 1：打开任务计划程序

1. 按 `Win + R`
2. 输入 `taskschd.msc`
3. 按回车

#### 步骤 2：创建基本任务

1. 点击"创建基本任务"
2. 名称：`抖音热榜评论爬取`
3. 描述：`每 2 小时自动爬取热榜视频评论`
4. 触发器：选择"一次"
5. 开始时间：设置为当前时间
6. 操作：选择"启动程序"
7. 程序/脚本：`python`
8. 添加参数：
   ```
   scheduler_hot_comment.py --interval 2 --video-count 10
   ```
9. 起始于：`d:\Project_graduation\douyin-main`

#### 步骤 3：配置重复执行

1. 创建完成后，双击任务
2. 在"触发器"选项卡，点击"编辑"
3. 勾选"重复任务间隔"
4. 选择"2 小时"
5. 持续时间选择"无限期"
6. 点击"确定"

#### 步骤 4：配置运行设置

1. 在"常规"选项卡：
   - 勾选"不管用户是否登录都要运行"
   - 勾选"使用最高权限运行"
   - 配置为"运行时不显示 UI"
2. 在"条件"选项卡：
   - 取消勾选"只有在计算机使用交流电源时才启动此任务"
3. 在"设置"选项卡：
   - 勾选"如果任务运行时间超过以下时间，停止任务"：设置为 1 小时

## 🔧 故障排除

### 问题 1：无法启动

**错误：** `python: command not found`

**解决：**
1. 确保已安装 Python
2. 将 Python 添加到系统 PATH
3. 或使用完整路径：`C:\Python39\python.exe`

### 问题 2：Cookie 无效

**错误：** `请先在设置中配置 Cookie`

**解决：**
1. 在设置中更新 Cookie
2. 确保 Cookie 来自登录状态

### 问题 3：任务计划不执行

**解决：**
1. 检查任务计划程序中的任务状态
2. 查看"上次运行结果"
3. 检查日志文件
4. 确保起始目录正确

### 问题 4：爬取失败

**错误：** 日志中显示爬取失败

**解决：**
1. 检查网络连接
2. 更新 Cookie
3. 查看具体错误信息
4. 降低爬取数量
